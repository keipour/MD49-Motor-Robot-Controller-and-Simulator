
F1  -  Help (this page)
Esc -  Exit
F10 -  Stop robot
F12 -  Toggle control of the robot using the keyboard

When control is given to keyboard, these keys work:

Up arrow key       -  Move robot forward
Down arrow key  -  Move robot backward
Right arrow key   -  Rotate robot clockwise
Left arrow key     -   Rotate robot counter-clockwise
Shift key               -   Stop robot
